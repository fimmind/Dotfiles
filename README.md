## Installation
```shell
$ cd ~
$ git clone https://github.com/fimmind/Dotfiles
$ cd Dotfiles
$ make
```

## GTK theme
Use `lxappearance`:
* Widget: Plata-Noir
* Icon: Papirus
* Mouse cursor: Breeze
