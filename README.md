## Installation
```shell
$ cd ~
$ git clone https://github.com/fimmind/Dotfiles
$ cd Dotfiles
$ sudo make
```

## GTK theme
Use `lxappearance`.

Favorite themes:
* Widget: Plata-Noir
* Icon: Papirus
* Mouse cursor: Breeze
